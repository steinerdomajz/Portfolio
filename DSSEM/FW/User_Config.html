  1. Connect the device to your computer via a USB-to-UART adapter.
  2. Click the Connect via USB button and select the appropriate COM port.
  3. Fill in the configuration data (Device ID, APN, Server IP, etc.).
  4. Check the estimated battery life in the green panel.
  5. Reset the device to initialize serial communication.
  6. Click Save Configuration to Device to store parameters in the device's memory.
  7. You can monitor the configuration progress in the black terminal window.
